# Contributing

Thanks for helping improve Agent Sandbox Cookbook.

This project values practical examples, verified docs, and clear comparisons. If a contribution depends on a vendor SDK or runtime feature, include source links and the version or date you checked.

## Good Contributions

- A runnable agent tool example.
- A vendor SDK compatibility note.
- A sandbox failure-mode demo.
- A field note about a recent runtime change.
- A correction to an outdated API example.
- A side-by-side comparison across providers.

## Content Standards

- Prefer minimal runnable examples over large demos.
- Keep vendor-specific claims sourced.
- Mark speculative or unverified behavior clearly.
- Include exact commands when possible.
- Include safe failure examples, not only happy paths.
- Avoid leaking real API keys, account IDs, or private endpoints.

## Recipe Structure

Each recipe should include:

- Purpose
- Prerequisites
- How to run
- Tool schema
- Sandbox contract mapping
- Expected output
- Failure cases
- Notes and limitations

## Field Note Structure

Each field note should include:

- Date
- Vendor/runtime
- What changed
- Why it matters
- Sandbox impact
- Links
- Follow-up checklist
