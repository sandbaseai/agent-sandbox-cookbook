# Field Notes

Short notes on recent agent runtime and sandbox-related changes.

Use one file per update:

```text
YYYY-MM-DD-vendor-topic.md
```

Each note should include:

- What changed
- Why it matters for sandboxed tools
- What example or checklist should be updated
- Source links
- Confidence level

Field notes should be concise and practical. The goal is to help developers decide whether their agent runtime integration needs to change.
